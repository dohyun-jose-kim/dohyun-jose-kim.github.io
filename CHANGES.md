# Blog v0.2.3 — 인터랙션 개선 내역

`react-blog-v0.1.6` → `react-blog-v0.2.2` (DC: `Blog v0.2.2.dc.html`) 사이에 더해진 주요 인터랙티브 개선만 정리합니다. (콘텐츠/문구/폰트 교체 등 단순 변경은 제외)

---

## 1. Experience — 마스터·디테일 타임라인
- **고정 행 높이 + %-기반 축**: 행 높이를 고정(184px)해 펼치고 접을 때 세로 흔들림이 없음. 축(세로선)은 % 좌표라 폭이 바뀌어도 안 깨짐.
- **축 가로 이동**: 접힘 72% → 펼침 40%로 축이 미끄러지듯 이동(약 32% 횡이동)하는 것이 핵심 모션.
- **"날아가는" 소제목(detailTitle)**: 접힘 시 축 기준 상대좌표로 떠 있다가(우측 정렬), 펼치면 불릿 칼럼 상단으로 날아가 좌측 정렬로 전환.
- **축 노드 동기화**: 회색(비활성) 노드는 소제목 y에 맞춰 아래로, 활성 시 위로 — 소제목과 같은 y를 부드럽게 추적.
- **불릿 cascade**: 펼칠 때 우측에서 순차(`0.12 + i*0.04 + j*0.07s`)로 페이드·슬라이드 인.
- **인턴십 둥지 구조**: 인코인턴십을 인실리코젠 연구원 항목 아래 들여쓰기 자식 행(↳ 커넥터, 구분선 제거, 제목 축소)으로 묶음.
- **제목 줄바꿈**: 모든 경력 제목을 "·" 기준으로 자동 줄바꿈.

## 2. Life — 플립 카드 그리드
- **3D 플립 + 중심 당김(center-pull)**: 클릭 시 카드가 Y축으로 뒤집히며 확대. transform-origin을 페이지 중심 쪽으로 잡아 안쪽으로 당겨지는 느낌.
- **열 수 적응형 확대**: 실제 렌더된 열 수(3/2/1)를 측정해 확대 배율을 차등 적용(3열 2배 · 2열 ~1.55배 · 1열 ~1.06배) — 전체폭 카드가 화면 밖으로 튀지 않음.
- **호버 모티프 애니메이션**: 카드별 라인 아이콘이 호버 시 dim→accent로 바뀌고 살짝 떠오름.
- **카드 뒷면 재구성**: 제목(위) · 사진 슬롯(가운데, 드래그로 채움) · 상세(아래) 3단 배치.
- **카드 통합/재배치**: 승마·요트·등산을 한 카드(아이콘 3개 나란히)로 합치고, 마지막 칸은 "+" 자리표시.

## 3. 반응형 (미디어쿼리 없이 JS 측정 기반)
- **사이드바 접힘**: 콘텐츠 폭 720px 미만이면 좌측 사이드바가 사라지고 본문이 전체 폭 사용. 햄버거 버튼 → 슬라이드인 오버레이(배경/✕/Esc/링크 클릭으로 닫힘).
- **Experience 스택 전환**: 칼럼 폭 640px 미만이면 절대 위치 타임라인 대신 세로 스택 아코디언으로.
- **Life 그리드 단계 축소**: `auto-fit` + `minmax`로 3열 → 2열 → 1열.
- **Hero 컨테이너 스케일**: `cqw` 단위로 콘텐츠 폭 기준 스케일 → 사이드바 폭을 감안해 제목이 줄어듦.

## 4. 전역 인터랙션
- **테마/언어 토글**: 다크/라이트, KO/EN 전환 + `localStorage` 저장·복원.
- **스크롤 연동 내비**: 현재 섹션을 감지해 좌측 TOC의 바/라벨 활성 표시.
- **커서 스포트라이트**: 다크 모드에서 포인터 위치를 따라가는 방사형 그라데이션(`--mx/--my`).
- **prefers-reduced-motion 존중**: 모션 최소화 설정 시 모티프 애니메이션 비활성.

## 5. DC 전용 — 라이브 튜닝 prop
DC 버전은 다음을 Tweaks 패널 prop으로 노출해 실시간 조정:
- 폰트(영문 enum / 한글 enum), 테마 기본값, 강조색(다크/라이트)
- 플립 배율, center-pull
- Experience 좌표 일체(축 위치, 행 높이, 레일 폭/이동, 소제목 폭·위치, 디테일 갭/크기, 제목 크기 등)

> React 버전(`react-blog-v0.2.3`)은 위 prop들을 `App.tsx` 상단의 `EXP` / `FLIP_SCALE` / `CENTER_PULL` 상수로 고정해 두었습니다. 사진 슬롯은 plain React에선 자리표시(placeholder)로, DC/standalone에선 드래그-드롭 슬롯으로 동작합니다.

# v0.2.3 — 추가 개선

- **스크롤 등장 애니메이션**: 각 섹션의 콘텐츠가 뷰포트에 들어올 때 살짝 떠오르며(translateY 22px → 0) 페이드인. `prefers-reduced-motion` 설정 시 비활성. (`App.tsx`의 scroll-reveal `useEffect`)
- **폰트/번들 정리**: 미사용 폰트 9종(Manrope·Space Grotesk·Sora·Plus Jakarta Sans·IBM Plex Sans KR·Gowun Dodum·Nanum Gothic …)과 Pretendard CDN 제거 → 실제 사용하는 **Inter · Noto Sans KR · IBM Plex Mono**만 로드 (`index.html`). `index.css`의 미사용 keyframes(`streak`/`eq`/`blink`/`nudge`)도 삭제.
- 버전 라벨 v0.2.3로 갱신.
